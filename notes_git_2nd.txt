Command
-----------------
Review untracked file: git status 
Adding new file: git add <file Name>
Adding multiple file: git add -A
Commit to branch: git commit -m "commit comments"
Commit Mulitple file into branch: git commit -a -m "commit comments"
Verify branch update: git log
push change to Master: git push origin master
Update the current enviroment: git pull origin master
